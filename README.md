# SLA Performance Dashboard

## Overview
This project analyzes Service Level Agreement (SLA) performance across carriers, routes, and shipping modes.

## Key Metrics
- SLA Breach Rate
- On-Time Delivery Rate
- Average Delay (Days)
- Average Shipping Cost
- Cost vs Reliability Comparison

## Tools Used
- Power BI
- DAX
- Data Cleaning & Transformation

## Key Insights
- Identified high-risk carriers
- Analyzed route-level delay patterns
- Compared cost impact between breached and on-time shipments
